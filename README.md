Project Overview:
This project implements a relational database for an e-commerce platform.
It manages Customers, Products, Categories, Orders, Order Items, and Payments.
The database supports real-world operations such as placing orders, tracking payments, and generating reports.

Description
This project is a relational database system for an e-commerce platform.

It allows:
- Managing customers
- Managing products and categories
- Placing orders
- Processing payments
